**python-profeld** 是一个生存分析、预测性维护python工具。

ProFeld是戴姆勒内部开发使用的Weibull生存分析软件名字，python-profeld则配备了更多相关算法。中文名扁鹊，希望做到故障领域的望闻问切。

## 快速入门

 预测性维护是工业互联网最广泛的应用之一。
